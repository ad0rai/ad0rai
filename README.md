## 안녕하세요, 김재현(Adora)입니다 👋

**넥스트증권**에서 Ambassador Program을 맡고 있습니다.

넥스트증권은 미국주식 거래에 특화된 증권사로, 국내 최초 에이전틱 브로커리지를 
지향하고 있습니다. API trading에 진심인 팀입니다.

---

### 🚀 지금 하고 있는 것

넥스트증권 미국주식 Open API 서비스 런칭을 준비하고 있습니다.

REST API 기반으로 자동매매, 알고리즘 트레이딩, 에이전틱 트레이딩까지 가능한 
환경을 만들고 있어요. 현재 9월 CBT 런칭을 목표로 개발 중입니다.

저는 개발보다는 커뮤니티와 생태계를 담당하고 있습니다. API trading을 직접 하는 
분들과 연결되고 싶어서 GitHub을 시작했어요.

---

### 🤝 함께할 분들을 찾고 있습니다

넥스트증권과 함께 성장할 **Ambassador**를 모집하고 있습니다.

저희는 작은 팀입니다. 빠르게 움직이고, 불편한 점을 말씀해 주시면 실제로 바뀝니다.

Ambassador분들께는 실거래 우대 수수료는 기본이고, 그보다 더 중요한 것들을 
드리고 싶습니다.

- 엔지니어팀과 직접 소통할 수 있는 채널 (Discord · Slack)
- 제품 로드맵을 함께 설계하는 경험
- 글로벌 컨퍼런스 참여 기회

대형사가 구조상 줄 수 없는 것들입니다.

API trading을 직접 하시는 분, 관련 콘텐츠나 커뮤니티를 운영하시는 분이라면
